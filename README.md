# FlappyBird-test-
I created a Flappy Bird style game using HTML, CSS, and JavaScript, working with the HTML5 canvas.
I built a full game loop, added images to the canvas, and implemented click handlers so the bird can jump.
I also generated pipes randomly and animated them moving across the screen, added collision detection between the bird and the pipes, and created a running score system.


Реализовала игровой цикл, добавила изображения на холст и сделала обработчики кликов, чтобы птичка могла подпрыгивать.
Также я настроила случайную генерацию труб и их движение по экрану, добавила проверку столкновений между птичкой и трубами и систему подсчёта очков.
